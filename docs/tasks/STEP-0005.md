# Task Card — STEP-0005

- **ID:** STEP-0005
- **Type:** feature
- **One outcome:** An elevated dark-premium, Apple-minimal **design system** (tokens + shared shell components) applied to the **home page**, establishing the visual language the rest of the site will inherit.

(3) Home page redesigned (cinematic hero + refined featured grid) using the system; product cards elevated with tasteful hover.
