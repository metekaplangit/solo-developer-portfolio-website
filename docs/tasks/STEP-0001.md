# Task Card — STEP-0001

- **ID:** STEP-0001
- **Type:** feature
- **One outcome:** A validated content model (Astro content collections + Zod schemas for Product, StoreLink, PrivacyPolicyEntry, MediaAsset) plus a static shell (base layout, nav, footer) whose generated routes — home, catalog, privacy, support/contact, about — render from sample content files.

- **Acceptance criteria:** (1) Deliberately invalid content **fails** `npm run build`/`astro check` with a clear error. (4) No backend, database, serverless, or third-party service is introduced (static-only guardrail holds).
