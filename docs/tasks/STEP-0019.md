# Task Card — STEP-0019

- **ID:** STEP-0019
- **Type:** feature (UI — premium craft polish)
- **One outcome:** Raise the site to a genuinely premium, perfectionist feel by adding the small-but-decisive craft details top design leaders use — tactile interaction feedback, typographic precision, and refined finishing touches — all within static / zero-JS / accessible constraints.
- **Why now:** User wants a higher, much more premium-quality website; research (Linear + micro-interaction sources) plus a code audit showed we had the foundation but lacked the "expensive-feeling" micro-details.
- **Research basis (lean, non-agent):** Linear ([linear.app](https://linear.app)) + micro-interaction sources ([DesignRush](https://www.designrush.com/best-designs/websites/microinteractions), [Webflow](https://webflow.com/blog/microinteractions)): premium = interaction feedback (hover **and** press), mathematical precision/consistency, typographic craft (no orphans), restraint + performance.
- **Acceptance criteria:** (1) **Interaction feedback:** buttons get a press (`:active`) settle + a crafted top-edge highlight + hover glow; cards get a hover edge-highlight. (2) **Animated content-link underlines** that slide in on hover (reading areas, footer, back/section links; nav/buttons/cards keep their own states). (3) **Typographic precision:** `text-wrap: balance` on headings, `pretty` on paragraphs, `tabular-nums` on dates. (4) **Finishing details:** accent-tinted `::selection`, subtle on-theme custom scrollbar (desktop), smooth `scroll-behavior`. (5) **Scroll-reveal consistency:** about / support / privacy sections animate in like the rest. (6) Everything CSS-only, zero-JS, **reduced-motion-safe**; contrast/focus preserved; no horizontal overflow desktop/mobile. (7) `npm run build`, `npm run check`, `npm test`, validator pass.
- **Non-goals:** Kinetic/JS headline animation, cursor effects, redesign, new sections/pages.
- **Evidence method:** command/log (build/check/test/validator) + runtime computed-style verification of each detail + overflow scan (desktop + mobile) + before/after screenshot.
- **Proof classification:** machine-verifiable + manual-runtime (craft is human-judged).
- **Required checks:** `npm run build`; `npm run check`; `npm test`; validator; runtime visual + overflow scan.
- **Issue links:** none. (a11y is guarded by the STEP-0017 Lighthouse gate.)
- **Rollback/containment:** work on `feature/STEP-0019-premium-craft`; revert the merge or branch from `v0.18.x`; CSS/token + component style only.
