# Task Card — STEP-0017

- **ID:** STEP-0017
- **Type:** feature (tooling / hardening) — resolves **LEDGER-002**
- **One outcome:** Add an **automated accessibility + quality gate** (Lighthouse CI) that runs on the built pages in CI and on deploy, failing on accessibility regressions — closing the manual-only gap that let the `--faint` contrast bug through (AUDIT-0003-F2 / MC-0008).
- **Why now:** User asked for it after AUDIT-0003 + MC-0008 both recommended it; extra relevant now that scroll motion is in.
- **Design:** `@lhci/cli` (dev-only) runs `lhci autorun` against `staticDistDir: ./dist`. `lighthouserc.json` asserts **`categories:accessibility` ≥ 0.95 as an error** (deterministic), and performance / SEO / best-practices as **warnings** (informational — CI-runner variance never fails the build). Reports written to `.lighthouseci/` (filesystem target — no third-party upload). ubuntu-latest's bundled Chrome launches it.
- **Acceptance criteria:** (1) `lighthouserc.json` + `npm run lhci` script. (2) CI (`ci.yml`) gains an `accessibility` job (runs on push + PR). (3) The **deploy** (`deploy.yml`) runs the same gate after build, before publish — a regression blocks the live deploy. (4) The gate **passes on the current pages** (verified in real CI via a PR before merge). (5) Zero-cost / GitHub-only / static preserved — no third-party service; dev-only dependency does not ship. (6) `npm run build`, `npm run check`, `npm test`, governance validator pass; **production** `npm audit --omit=dev` = 0 (dev-tool advisories don't ship).
- **Non-goals:** Gating on performance scores (flaky); visual-regression snapshots; changing page design.
- **Evidence method:** command/log (build/check/test/validator + YAML/JSON parse) + **real CI run** of the `accessibility` job (green) via a PR; Lighthouse accessibility score recorded.
- **Proof classification:** machine-verifiable (the CI job is the proof).
- **Required checks:** `npm run build`; `npm run check`; `npm test`; validator; **Lighthouse CI accessibility job green in GitHub Actions**.
